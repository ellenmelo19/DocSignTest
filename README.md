# Teste Técnico SuperSign - Desenvolvedor(a) FullStack

Este repositório contém a solução para o teste técnico da SuperSign, implementando uma API em Node.js com Fastify e uma interface web em Next.js para gerenciamento de documentos.

## Estrutura do Projeto
- `/backend`: API com Node.js, Fastify, Prisma e PostgreSQL
- `/frontend`: Interface web com Next.js

## Tecnologias Utilizadas
- Backend: Node.js, Typescript, Fastify, Prisma.
- Frontend: Next.js, Typescript, Tailwind CSS.
- Banco: PostgreSQL (via Docker)
- Arquitetura: Clean Architecture com separação domain/application/infra/presentation.
- Diferenciais: DTOs, Testes automatizados (Vitest), Boas Práticas REST.
